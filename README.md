# React + SCSS Project

## Overview

This project is built with **React** and **SCSS**.  
It includes all source files in the `src/` folder and a production-ready build in the `build/` folder.

---

## Running the App

This project is built with React and can be run directly in CodeSandbox or locally on your machine.


## Running in with CodeSandbox

* The app will run at: [RainFocus Summit ](https://w52r37-3000.csb.app/)


## How to Run Locally

1. Clone the repo:

    `git clone https://github.com/kedinger1178/rainfocus.git`

2. Navigate to the project folder:

    `cd rainfocus`

3. Install dependencies:

    `npm install`

4. Start the development server:

    `npm start`

5. Start the Sass compiler:

    `npm run sass`

* The app will run at: http://localhost:3000
* Changes to SCSS or React files will automatically update in the browser.
